# PM Tool

A project management portal (NestJS + React/TypeScript/Tailwind) covering projects, tasks, time tracking, timesheets, invoices, financials, and reports across internal, vendor, and client roles.

- **Backend:** `backend/` (NestJS + Prisma + PostgreSQL) — see `backend/README.md`
- **Frontend:** `frontend/` (React + Vite + Tailwind) — see `frontend/README.md`
